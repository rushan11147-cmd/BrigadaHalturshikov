# Бригада халтурщиков

Funny physics co-op (Godot 4) — бригада выполняет заказы: переезды, ремонт, перенос мебели.

## Запуск

1. Открыть проект в **Godot 4.7+**
2. F5 → меню → выбрать сложность → Играть

## Управление

| Клавиша | Действие |
|---------|----------|
| WASD | Ходьба |
| Shift | Бег |
| Мышь | Обзор |
| Space | Прыжок |
| E | Взять / отпустить |
| ЛКМ | Бросить |
| R | Заново |
| Esc | Пауза / меню |

## Структура

- `scenes/` — игрок, объекты, Main
- `scripts/generation/` — процедурная генерация
- `data/` — комнаты, предметы, миссии, события (добавляй `.tres` без правки кода)
- `assets/models/player/` — Food Worker (J-Toastie)
